### ✨ Что такое YoLauncher?
**YoLauncher** — это проект, рожденный из чистого энтузиазма. Мы — группа ребят, которым просто по кайфу делать удобный и крутой лаунчер для Minecraft.

* **☕ Для души:** Мы не корпорация, у нас нет офисов и скучных галстуков. Только код, чай и желание сделать проект «для своих».
* **🕊️ Свобода:** Мы верим в открытый код и сообщество. Наша цель — дать игрокам инструмент, который работает быстро и без лишнего мусора.
* **🤝 Твое участие:** Мы рады всем, кто хочет помочь кодом, идеями или просто добрым словом.

### 🛠 Основные фишки 
* **👾 Pixel Aesthetic UI:** Полный редизайн в стиле 8-бит (1150x700).
* **🗂️ Instance System:** Создавай отдельные сборки со своими настройками и RAM.
* **🎭 Skin Support:** Загрузка и 2D-превью твоих PNG скинов.
* **🚀 Smart Launch:** Исправленный прогресс-бар и поддержка Forge/Fabric/OptiFine.
* **🔌 Offline Mode:** Корректная работа UUID и мультиплеера без интернета.

### 🛠 Разработка (Build from source)
Если хочешь собрать лаунчер сам или внести правки:

```bash

cd yolauncher

# Установить зависимости
